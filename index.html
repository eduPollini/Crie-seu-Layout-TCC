Dicas

Selecionar

Clique no botão selecionar, para escolher o tipo de item ao lado.

Mover

Clique com o botão esquerdo do mouse, em um dos itens já posicionados, para movê-lo.

Cancelar

Clique com o botão direito do mouse, enquanto entiver com um item selecionado, para cancelar a seleção.

Apagar

Clique com o botão direito do mouse, em um dos itens já posicionados, para movê-lo.

Ver Foto

Coloque o cursor sobre a imagem de um dos itens, no menu de itens, para vizualisar sua imagem real.

Rotacionar

Clique no botão , para rotacionar os itens que serão inseridos. Ou então utilize as teclas e .

Numerar

Os itens podem ser numerados, selecionando no botão . Para insererir um número utilize o clique esquerdo, já para remover, o clique direito.

Recomeçar

Clique no botão , e depois em confirmar, para apagar todos os itens posicionados.

Baixar

Clique no botão download, para baixar o mapeamento como imagem (.png). Você será redirecionado para o formulário de contato.

Enviar

Clique no botão enviar, para mandar seus dados via e-mail, seu mapeamento será anexado automáticamente.
